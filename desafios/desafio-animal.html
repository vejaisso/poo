Desafio - Classe Animal e Herança

Curso Técnico Integrado ao Ensino Médio - IFSP Campinas

Descrição do Desafio

Neste exercício, você deverá trabalhar com herança e polimorfismo para criar diferentes tipos de animais que se comunicam de formas distintas.

Instruções

  • Crie uma classe Animal com um método abstrato fala().
  • Crie as classes Homem, Cão e Gato, herdando de Animal.
  • Redefina o método fala() em cada classe para retornar, respectivamente: "Oi", "Au au" e "Miau".
  • Crie um vetor com 10 elementos do tipo Animal, instanciando objetos de Homem, Cão e Gato.
  • Utilize um loop para percorrer o vetor e fazer cada animal falar.
Dica: Utilize herança para evitar repetição de código e explore o polimorfismo para chamar o método fala() de forma genérica.

Objetivo

O objetivo deste desafio é reforçar os conceitos de classes abstratas, herança e polimorfismo, aplicando-os em um contexto simples e prático.
