Orientações para Desenvolvimento

Para realizar os desafios, utilize uma IDE ou editor de código com suporte à linguagem Java. As opções recomendadas são:

  • NetBeans – Interface simples e fácil para iniciantes.
  • IntelliJ IDEA – Completo e com recursos avançados para projetos maiores.
  • Visual Studio Code – Leve e flexível, com suporte a Java via extensões.

Escolha a ferramenta que você mais se adapta e lembre-se de organizar bem seus arquivos e pacotes para facilitar a execução e manutenção do código.
